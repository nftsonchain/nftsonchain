# NFTs OnChain

NFTs OnChain is a cross-chain NFT discovery hub where users can explore, learn about, and submit NFT collections across multiple blockchains.

## Features

* Cross-chain NFT discovery
* Search and filter NFT collections
* Like and favorite NFT collections
* User authentication with Clerk
* AI-powered NFT education assistant
* NFT submission system
* User profiles and saved favorites

## Supported Chains

* Ethereum
* Solana
* Bitcoin
* Gram
* Polygon
* Base
* Cardano
* Cosmos
* XRP
* More coming soon

## Tech Stack

* Next.js 16
* TypeScript
* Tailwind CSS
* MongoDB
* Clerk Authentication
* OpenAI API

## Local Development

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

http://localhost:3000

## Deployment

Deployed with Vercel.
